# 🏁 RetroCycles: Ilona's Competitive Mod Pack

Welcome to the professional mod pack for RetroCycles (0.2.9.3.0). This mod is designed for competitive players who need better situational awareness and precise feedback.

## 📥 How to Download
To get the latest version of the mod, go to the **[Releases](https://gitlab.com/ilonamoer/0-2-9-3-0-modpack/-/releases)** page and download the `.zip` file from the latest release.

## 🛡 Trust & Authorship
- **Transparent Source:** Every line of code is open. You can see exactly what changed compared to the official version in the [Commit History](https://gitlab.com/ilonamoer/0-2-9-3-0-modpack/-/commits/ilonas-mod).
- **Authorship:** Copyright (c) 2026 Ilona. Licensed under GPL v2+.
- **Fair Play:** The client identifies itself as `0.2.9.3.0-ilonas-modpack` for tournament compliance.

## ⚙️ Installation
1. Download the latest `.zip` from the **Releases** section.
2. Extract the archive.
3. Replace your original `armagetronad_main` binary (usually located in your Steam folder under `usr/bin/`).
4. Detailed instructions are included in the `README.txt` inside the archive.
