# 🏁 RetroCycles: Ilona's Competitive Mod Pack

Welcome to the professional mod pack for RetroCycles (0.2.9.3.0). This mod is designed for competitive players who need better situational awareness and precise feedback.

## 📥 How to Download
To get the latest version of the mod, go to the **[Releases](https://gitlab.com/ilonamoer/0-2-9-3-0-modpack/-/releases)** page and download the `.zip` file from the latest release.

## 🌟 Key Features
- **Cut-off Aimbot:** Intelligent HUD prompts (`<<< CUT` or `>>> NO`) that calculate if you can successfully cut off an enemy, factoring in both your ping and theirs.
- **Interactive Minimap:** Dynamic rendering of arena zones (Sumo/Fortress) with full rotation support—see exactly where the safe zone is at all times.
- **Rubber Usage Gauges:** Visual indicators above all players showing their real-time rubber consumption.
- **Proximity Danger Warning:** A subtle screen-edge red glow that activates when an enemy cycle is dangerously close.
- **Zone Collapse Timers:** Accurate countdown timers for arena contractions.
- **Rubber Battery:** A persistent, clean gauge for your own rubber resources.

## ⚙️ Installation
1. Download the latest `.zip` from the **Releases** section.
2. Extract the archive.
3. Replace your original `armagetronad_main` binary (usually located in your Steam folder under `usr/bin/`).
4. Detailed instructions are included in the `README.txt` inside the archive.
